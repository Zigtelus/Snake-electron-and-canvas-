# О ПРОЕКТЕ <br>
Данный проект создается в тренировочных целях.<br>
При помощи этого проекта я планирую лучше понять Electron и Canvas<br>
Проект использует мою собственную сборку WEBPACK.<br>

# ТЕХНОЛОГИИ <br>
- React 18
- Function components
- React-router-dom 6
- React-dom 18
- React-redux 9
- Styled component 6
- SASS 1.69
- TypeScript 5
